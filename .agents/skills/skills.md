# Skills Index

Reusable engineering workflows for the portfolio project.
Each skill lives in its own folder as `SKILL.md` with YAML frontmatter.

## Content & Data

| Skill | When to use |
|---|---|
| [add-database-table](add-database-table/SKILL.md) | New SQLite table for a new entity |
| [add-zod-schema](add-zod-schema/SKILL.md) | New Zod schema for DB rows or API inputs |
| [add-db-query](add-db-query/SKILL.md) | New query function for reading from SQLite |
| [database-migration](database-migration/SKILL.md) | Altering schema in production without data loss |

## API & Backend

| Skill | When to use |
|---|---|
| [add-trpc-router](add-trpc-router/SKILL.md) | New tRPC domain, typed CRUD, admin mutations |
| [add-protected-mutation](add-protected-mutation/SKILL.md) | Admin write operation requiring JWT auth |
| [add-api-route](add-api-route/SKILL.md) | Non-tRPC API route (file download, webhook, etc.) |

## UI & Frontend

| Skill | When to use |

| [update-translations](update-translations/SKILL.md) | Modifying or removing existing translation keys |
